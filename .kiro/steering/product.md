# Product Overview

Jazz Chord Practice App - An interactive web application for practicing jazz guitar chord shapes and fingerings.

## Core Features

- **Chord Practice Sessions**: Timed practice sessions with randomized chord problems
- **Visual Fretboard**: SVG-based guitar fretboard diagrams showing chord fingerings
- **Performance Tracking**: Session data collection with CSV export and analysis
- **Customizable Practice**: Options to filter by key, chord type, and string sets
- **Results Analysis**: Identifies problem areas and suggests chords to drill

## Target Users

Guitar players learning jazz chord voicings and wanting to improve their chord recognition and muscle memory.

## Key Value Proposition

Provides structured, measurable practice for jazz guitar chord shapes with immediate visual feedback and performance analytics.
